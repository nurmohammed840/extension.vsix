# Just Black

This dark theme was designed specifically for syntax highlighting.

This theme is not opinionated. If you think you can improve it, please open a [pull request](https://github.com/nurmohammed840/extension.vsix).

## Note

Add the following to your VS Code `settings.json`:

```json
{
    "editor.semanticHighlighting.enabled": "configuredByTheme"
}
```
